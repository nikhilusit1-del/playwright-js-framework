# Playwright UI Automation Framework

## Objective
Create an end-to-end UI automation framework using Playwright with JavaScript to validate login functionality.

## Tech Stack
- Playwright
- JavaScript
- Playwright Test Runner

## Framework Features
- Page Object Model (POM)
- Data-driven testing
- Positive & Negative login scenarios
- Error message validation
- Cross-browser testing
- Parallel execution
- HTML reporting
- Screenshots & videos on failure
